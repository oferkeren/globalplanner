# Global Travel Planner V11

גרסה גלובלית להפצה של מתכנן הטיולים.

## מה יש בפנים
- שאלון ראשוני לבניית טיול
- רינדור דינמי של דשבורד, לו״ז, מלונות ואטרקציות
- העלאת מסמכים מקומית
- מודול עבודה מרחוק כללי בלבד
- ייצוא וייבוא JSON
- שמירה ב-localStorage
- ללא נתונים אישיים, ללא דרכונים, ללא IP, ללא שמות משפחה

## הפצה
מתאים ל-Vercel / Netlify / GitHub Pages כאפליקציה סטטית.
אין צורך ב-ENV בשלב הזה.

## מגבלה חשובה
העלאת מסמכים בגרסה סטטית שומרת metadata בדפדפן בלבד. להפצה אמיתית עם משתמשים וקבצים קבועים צריך backend/storage כמו Supabase, Firebase או S3.
